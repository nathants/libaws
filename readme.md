# cli-aws

## why

aws should be simple and easy.

## what

tooling to make aws simple and easy.

## install

```
go install github.com/nathants/cli-aws@latest
```

## usage

```
cli-aws -h
```

## bash completion

```
source completions.d/cli-aws.sh

```
